The Apache Software Foundation is pleased to announce Apache Forrest 0.6
http://forrest.apache.org/mirrors.cgi

Apache Forrest (http://forrest.apache.org/) is an XML standards-oriented
project documentation framework built upon Apache Cocoon, providing XSLT
stylesheets and schemas, images and other resources. Forrest uses these
to render the XML source content into a website via command-line, robot
or as a dynamic web application.
