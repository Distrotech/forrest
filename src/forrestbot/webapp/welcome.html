Forrestbot's Dodgy Web Interface

This site contains automatically updated and built versions of websites built with the latest Forrest. It aims to provide a service similar to Gump, but for websites. Host projects' XML is automatically checked out and the HTML rebuilt every 4 hours.

In addition, if you have just made a site update and are itching to see the result, site refreshes can be triggered manually.

Certain users can also trigger an automatic publish of the generated HTML/PDFs to Apache's site CVS module (xml-site, avalon-site, etc).

To request a username/password that allows you to refresh and/or publish your project's site, please mail Jeff Turner.

If you have any questions, please mail Jeff Turner, or forrest-dev
